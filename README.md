# Poster2Genre - Классификация жанра фильма по постеру

## Описание

Проект решает задачу многоклассовой классификации жанра фильма по изображению постера с использованием глубокого обучения. В основе - свёрточная нейронная сеть на базе предобученной **ResNet50**, дообученная на коллекции постеров с жанровой разметкой.

**Гипотеза:** жанр фильма несёт устойчивую визуальную сигнатуру - цветовую палитру, композицию, образы персонажей, типографику - которую можно формализовать и извлечь с помощью CNN.

## Архитектура

- **Backbone:** ResNet50 (предобученный на ImageNet, дообученный)
- **Классификационная голова:** Linear → BatchNorm → ReLU → Dropout → Linear(num_classes)
- **Оптимизатор:** AdamW с дифференцированным lr (backbone: 5e-5, голова: 5e-4)
- **Scheduler:** CosineAnnealingLR
- **Размер входа:** 432×288
- **Эпохи:** 35 (ранняя остановка по val macro-F1), batch size 32
